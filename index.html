02 — Domain

Research Domain

Exploring the intersection of wearable IoT, machine learning, and child safety to build a comprehensive real-time monitoring ecosystem.

Background

Child safety in modern environments demands intelligent, non-intrusive monitoring systems. Parents and guardians increasingly rely on digital tools, yet existing solutions lack integration between physiological health monitoring, location awareness, and behavioral threat detection. The convergence of wearable technology, edge computing, and machine learning creates an unprecedented opportunity to deliver a unified child safety platform through an everyday smartwatch.

"Existing child monitoring tools operate in silos — health apps, GPS trackers, and audio monitors function independently, leaving critical cross-domain patterns undetected."

Research Gap: No current smartwatch solution integrates dehydration early-warning with adaptive geo-fencing, privacy-preserving audio analysis, and anxiety-aware coaching in a single cohesive platform tailored for children aged 5–14.

Research Problem

How can a smartwatch-based system effectively integrate multi-modal health and safety monitoring — including dehydration detection, geospatial awareness, audio-based threat identification, and psychological health coaching — while preserving children's privacy and maintaining real-time responsiveness?

Research Objectives

1

Develop a pediatric dehydration early-warning algorithm using biomarker data from smartwatch sensors.

2

Build an adaptive geo-fencing system that responds to context and behavioral patterns.

3

Implement privacy-preserving audio classification for stranger/threat detection on-device.

4

Design an anxiety-aware coaching module providing real-time safety and health guidance.

Methodology

A design science research methodology combining literature synthesis, prototype iteration, and empirical validation with simulated and real-world child safety scenarios.

  • Sensor data collection and preprocessing
  • ML model training and on-device deployment
  • Federated learning for privacy preservation
  • Mobile-integrated alert and coaching UI
  • Usability testing with target age group

Technologies Used

A modern full-stack IoT and ML technology stack enabling real-time processing, mobile alerts, and secure cloud synchronisation.

TensorFlow Lite Python Flutter Firebase Google Maps API Federated Learning Wear OS SDK Node.js MQTT OpenAI Whisper FastAPI MongoDB

Individual Research Components

Pediatric Dehydration Early-Warning

Machine learning model analysing heart rate variability, skin conductance, and activity data to predict dehydration onset in children before clinical signs appear. Real-time alerts to parent mobile app.

Adaptive Geo-Fencing & Location Tracking

Context-aware geospatial monitoring that dynamically adjusts safe-zone boundaries based on time-of-day, school schedules, and known behavioural patterns — reducing false alerts for parents.

Privacy-Preserving Audio Threat Detection

On-device audio classification using federated learning to identify threatening voice patterns and stranger interactions, without transmitting raw audio to the cloud — preserving child privacy.

Anxiety-Aware Safety & Health Coaching

Adaptive coaching engine that detects stress and anxiety biomarkers, delivering contextual safety reminders, breathing exercises, and hydration nudges tailored to the child's current state.
